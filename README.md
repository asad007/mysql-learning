# MySQL Learning Journey 🗄️

A structured MySQL learning project built from scratch using MySQL Workbench.

## 📚 What this covers
- Week 1: SQL foundations — CREATE, INSERT, SELECT, UPDATE, DELETE
- Week 2: Deeper querying — AND, OR, LIKE, GROUP BY, aggregate functions
- Week 3: Multi-table relationships — JOIN, FOREIGN KEY, UNION
- Week 4: Real-world skills — Subqueries, string functions, dates, indexes

## 🛠️ Tools used
- MySQL Workbench
- MySQL 8.0

## 📁 Structure
Each week has its own folder with a `.sql` file for every day.
Mini projects are included at the end of each week.

## 🎯 Goal
Build practical SQL skills through consistent daily practice,
progressing from zero to real-world database design.
